These generic codes were developed by Mr. Jan Axelson. These were modified for this project
## Host Side: 
  "generic_hid_cs_50" Created with Visual Studio 2008 for the .NET Framework 2.0 or later. Uses Filestreams.
## Device Side: 
  "device-hid-generic.zip" is a modification of Microchip's USB example titled Device - HID - Custom Demos. This modification adds support for vendor-defined control transfers. Written for the PIC 18F4550 and Microchip's MPLAB mcc18 compiler v3.44 with the Microchip Libraries for Applications v2012-10-15. Tested with MPLAB v8.88 and MPLAB X v1.60.
